Quick Start

Ask what they want to do. Give dropdowns for "Personal Round", "Join Tournament"

Personal Round »

Playing by yourself. You can also add a user to your scorecard. So, if your playing with one or more friends in a foursome, you can see each others scores here. Uses the roundtype - single format.

Create a Tournament - Let's this user control the tournament details. Invite people to join the tournament. Give them a search bar that allows them to quick filter users to add to tournament. Also, let users join a tournament and they will show up as pending. The person starting tournament can approve them to join.

Manage active tournament - Add or remove players from the current tournament.

Join a tournament - Send an invitation to join a pending tournament. Must be approved by creator.

TOUR EVENT - The tour will be managed by someone else. Tour consists of multiple tournaments.
